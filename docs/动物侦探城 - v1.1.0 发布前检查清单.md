## 2. 发布目标

v1.1.0 发布目标是完成内容扩展底座，而不是新增正式内容包。

发布前必须确认：

- registry 可描述当前默认内容包。
- Web 可通过 registry 和 URL 参数加载内容包。
- 学习记录按内容包隔离。
- 内容校验支持默认、指定包和全量包。
- 浏览器回归支持默认、指定包和全量包。
- 内容工厂审核模板和人工门禁已补齐。
