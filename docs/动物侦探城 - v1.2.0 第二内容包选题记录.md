## 2. 选题目标

为 v1.2.0 的第二内容包接入做选题确认。此文档记录候选方向、判断依据、前置门禁和当前接入状态。当前内容包已作为 `draft` 接入 registry，用于技术验证；它仍未通过人工内容审核、教研审核和儿童可读性审核。

第二内容包应优先验证：

- registry 和课程入口能承载多内容包。
- 新内容包仍能复用 `content-runtime` 和 `game-core`。
- 内容生产可以按审核清单推进。
- 默认除法内容包体验不退化。

建议 v1.2.0 将第二内容包候选暂定为：

```text
math.bsd.g2.s2.unit-2-direction-position@0.1.0
```

状态：

```text
draft
```

原因：

- 项目愿景、GDD、UI 规范和里程碑文档均提到后续验证“方向与位置”新单元。
- 该候选能真正验证课程切换入口和多内容包隔离能力。
- 该候选已按 AI draft 方式生成，用于验证第二内容包技术闭环；正式发布前必须完成教材拆解复核和人工审核。

## 5. 开工前必须确认

- 教材来源：北师大版数学二年级下册第二单元的具体页码或课时来源。
- 知识点范围：方向、位置、路线描述、相对位置等是否全部纳入。
- 题型边界：是否需要地图点击、路线排序、拖拽或方位选择。
