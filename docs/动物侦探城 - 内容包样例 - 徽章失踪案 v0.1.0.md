## 2. 文档目的

本文档不是 PRD，也不是纯剧情文档，而是一个可以转换为 JSON 内容包的结构化样例。

它用于验证：

- 教材节点如何映射到通用知识点。
- 知识点内容包如何组织题目、知识卡、线索和复习规则。
- 案件和关卡如何从知识点地图中编排出来。
- Boss 如何由多个知识点和能力点组合生成。
- 现有静态 MVP 内容如何迁移到结构化内容包。
