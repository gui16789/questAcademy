## 5. 快速运行建议

v0.2.0 开始推荐静态服务运行：

```bash
python -m http.server 4173 --bind 127.0.0.1
```

访问：

```text
http://127.0.0.1:4173
```

直接打开 `index.html` 可以作为 v0.1.0 兼容方式，但 v0.2.0 读取 JSON 后不保证 `file://` 可用。
