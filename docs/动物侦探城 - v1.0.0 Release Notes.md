# 动物侦探城 v1.0.0 Release Notes

## 版本信息

- 产品版本：v1.0.0
- Git tag：v1.0.0
- 发布日期：2026-05-22
- 发布负责人：项目负责人 + 技术负责人
- 对应 Milestone：v1.0.0
- 发布类型：Major

## 版本目标

将 v0.2.0 静态单页升级为 React + TypeScript + Vite 正式前端工程，保留当前除法内容包和学习闭环，不新增玩法、不新增案件、不接后端。

## 产品范围

### 本次包含

- React + TypeScript + Vite 学生端工程。
- `content-schema` 内容包类型。
- `content-runtime` 内容加载与查询。
- `game-core` 判分、解锁、Boss、错题、勋章规则。
- 学生端页面组件迁移。
- v0.2.0 localStorage 数据迁移到 v1.0.0。
- v1.0.0 浏览器回归脚本。

### 本次不包含

- 不新增玩法。
- 不新增案件。
- 不新增单元。
- 不新增教材版本。
- 不接后端。
- 不做账号系统。
- 不做家长端。
- 不做内容后台。
- 不做完整视觉重设计。

## 是否影响学习记录

- 是否影响 localStorage：是。
- 是否需要数据迁移：是，v0.2.0 -> v1.0.0。
- 是否存在数据丢失风险：当前回归未发现；localStorage 不可用时仍只保证当前 session 可体验。

## 回滚方案

- 回滚产品版本：回滚到 tag `v0.2.0`。
- 内容包版本：继续使用 `math.bsd.g2.s2.unit-1-division@0.1.0`。
- localStorage：v1.0.0 写入 `dataVersion: "1.0.0"`；如需回滚到 v0.2.0，应提示用户旧版页面可能无法完整读取 v1.0.0 新结构。
- 技术回滚：GitHub Release 标记异常，创建 patch 分支修复或回滚主分支到 v0.2.0 tag。
