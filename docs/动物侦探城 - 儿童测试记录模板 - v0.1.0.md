## 2. 文档目的

本文档用于记录《动物侦探城》MVP 儿童体验测试过程，帮助团队判断：

- 孩子是否能独立理解探案学习流程。
- 题目、线索、Boss 和错题复习是否适合二年级孩子。
- 孩子是否愿意继续体验。
- 家长是否认可学习价值。
- MVP 是否值得进入 v0.2.0 内容结构化和 v1.0 正式打磨。

## 3. 隐私与记录边界

### 3.1 匿名记录

儿童测试只使用匿名编号：

```text
child-001
child-002
child-003
```

### 3.2 禁止记录

不得记录：

- 真实姓名。
- 手机号。
- 家庭地址。
- 学校全名。
- 精确定位。
- 身份证件。
- 其他不必要的个人敏感信息。

### 3.3 可记录

可以记录：

- 年龄段。
- 年级。
- 是否接近二年级下册水平。
- 测试设备。
- 测试日期。
- 测试观察结果。
