## 3. v1.0.0 范围冻结

### 3.1 做

- React + TypeScript + Vite 工程化。
- content-schema 类型化。
- content-runtime 内容加载与查询。
- game-core 学习规则纯函数化。
- UI 组件化。
- localStorage 数据迁移。
- 构建、测试、内容校验和回归。

### 3.2 不做

- 不新增玩法。
- 不新增案件。
- 不新增单元。
- 不新增教材版本。
- 不接后端。
- 不做账号系统。
- 不做家长端。
- 不做内容后台。
- 不做完整视觉重设计。

## 4. 发布前验收预期

v1.0.0 发布前必须满足：

- `npm run check` 通过。
- `npm run test` 通过。
- `npm run validate:content` 通过。
- `npm run build` 通过。
- `npm run test:regression` 通过。
- 本地预览可运行。
- 当前内容包可加载。
- 4 个普通关卡可完成。
- 错题可记录和复习。
- 线索库、勋章馆可用。
- Boss 可解锁、重试和结案。
- v0.2.0 localStorage 可迁移。
- 移动端 320px 可用。
- 无 P0 / P1 Bug。
