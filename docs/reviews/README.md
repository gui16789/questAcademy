# 内容包审核记录

本目录用于保存每个内容包进入 registry 或发布前的审核记录。

命名格式：

```text
{contentPackageId}@{contentVersion}-review.md
```

每份审核记录应复制 `docs/动物侦探城 - 内容工厂交付清单与审核样例 - v0.1.0.md` 第 6-10 节，至少覆盖：

- 内容包审核总表。
- 内容审核记录。
- 教研审核记录。
- 儿童可读性审核记录。
- 问题记录。
- 发布结论。

第二内容包接入前，必须先在本目录留下对应审核记录。未关闭 P0 / P1 问题时，不得把内容包状态推进到 `approved` 或 `published`。
